# CSIE_Life_Simulator

## Instructions

To test the program locally, setting up a local HTTP server is necessary. (See [CORS policy](https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS))

With [Python3](https://www.python.org/downloads/) installed, execute:
```
 $ python3 server.py
```
